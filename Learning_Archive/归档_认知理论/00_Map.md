# 🗺️ Agent & Algorithm 核心架构全景映射图

欢迎来到极简化、只保留核心脑力引擎的特训。
要看穿一个 AI Agent 的心智架构，千万不要被花哨的前端或者数据库连接代码所转移注意力。

你只需要抓住一条流水线：**“感知 -> 判断 -> 回忆 -> 表达 -> 自省”**。

在 Aurora 中，这条流水线被极其严谨地拆分为四个核心 Python 文件。以下就是你进阶到顶级算法工程师必须要跨越的四座大山：

---

### 第一座山：感知与遗忘机制 (Memory Engine)
* **核心职责**：当外界输入一句话，系统需要把它翻译成数字几何距离进行测距（感知）。对于不重要的日常废话，系统必须有概率地将其永远遗忘（判断）。
* **对应源码**：`aurora/core/memory/engine.py`
* **主考算法**：`Embedding (多维向量嵌入)`, `VoI (Value of Information 打分机制)`, `Thompson Sampling (带概率的遗忘抽样)`
* **你的特训关卡**：👉 `01_Memory_Engine/`

### 第二座山：回忆迷雾与蝴蝶效应 (Graph Context)
* **核心职责**：当系统准备开口时，它必须去浩瀚的过往记忆中“打捞”相关的历史前情。如何确保打捞的不是生硬的关键词匹配，而是像人类做梦一样带有偶尔离奇但又符合因果律的跳跃？
* **对应源码**：`aurora/core/narrator/context.py`
* **主考算法**：`BFS (图论广度优先迷宫搜索)`, `Soft Threshold Pruning (图关系柔性概率修剪)`
* **你的特训关卡**：👉 `02_Graph_Context/`

### 第三座山：人格面具与演说投子 (Narrative Perspectives)
* **核心职责**：在正式向人类作答之前，Agent 需要选定一种语气或切入视角（人设切换）。如果死板地选用得分最高的视角，AI 就会像复读机。必须在这个环节引入极其关键的统计学分布。
* **对应源码**：`aurora/core/narrator/perspective.py`
* **主考算法**：`Beta Distribution (贝叶斯先验演化分布)`, `Softmax Function (神经网络概率极权归一化算子)`
* **你的特训关卡**：👉 `03_Perspectives/`

### 第四座山：暗中进化的纠错之手 (Metric Learning)
* **核心职责**：这是系统能在运营的漫长岁月里自我调整世界观的终极秘密武器。当用户的反馈证明系统之前的两段记忆“认错亲戚”了，系统如何在不重新拉通训练大模型的情况下，在线、极速、永久地修正自己在潜意识里的偏见矩阵？
* **对应源码**：`aurora/core/components/metric.py`
* **主考算法**：`Triplet Loss (三元组距离惩罚模型)`, `Gradient Descent (局部矩阵偏导数硬梯度下降)`
* **你的特训关卡**：👉 `04_Metric_Learning/`
